Absolutely. The site does not store your credit card or payment information. Payments are made via M-Pesa directly to our business number.

FIREBASE CONSOLE
Your database control center
🔥 Accessing Firebase

All site data lives in Firebase Realtime Database. To manage it, go to the Firebase Console.

Go to Firebase Console

Open your browser and visit: console.firebase.google.com

Select Your Project

Click on the project named mufasa-strength-0719

Open Realtime Database

In the left sidebar click Build → Realtime Database. All your members, trackers, songs, and settings are here.

ℹ️
Your Firebase project URL is: https://mufasa-strength-0719-default-rtdb.firebaseio.com

SECURITY CHECKLIST
Keeping your site & data safe
  • Secure your Firebase Console with a strong password and 2-Factor Authentication (2FA)
  • Set Firebase Realtime Database rules to restrict write access — only authenticated users should write
  • Never share your Firebase API keys publicly or post them on social media
  • Regularly export your members data to a spreadsheet as a backup
  • Keep your WhatsApp Business number separate from your personal number
  • If you notice unauthorised data in Firebase, immediately change your Firebase project rules
  • Enable Firebase App Check for extra protection against abuse
🔒 Recommended Firebase Database Rules
{
  "rules": {
    "testimonials": { ".read": true, ".write": "auth != null" },
    "videos": { ".read": true, ".write": "auth != null" },
    "songs": { ".read": true, ".write": "auth != null" },
    "data": { ".read": true, ".write": "auth != null" },
